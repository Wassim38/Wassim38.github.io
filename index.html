Mongo DB

Définition :

MongoDB est une base de données NoSQL orientée documents qui utilise des documents similaires à JSON avec des schémas dynamiques

Mini-Projet Elasticsearch et Kibana :

Pour ce projet, j'ai fait un compte rendu, disponible sur détails, qui montre les tâches suivantes :

  • Mettre en place une stack avec un cluster Elasticsearch associé à un serveur Kibana, et tout ça en local grâce à docker-compose
  • Créer un index Elasticsearch avec le bon nombre de shards afin que les données soient équitablement distribuées
  • Importer des données dans Elasticsearch en définissant un mapping adapté à ces données
  • Effectuer des recherches sur votre Elasticsearch, à la fois de données brutes et de données agrégées.
  • Configurer un Data View dans Kibana afin que les différentes données soient correctement affichées et gérées dans Kibana
  • Construire 2 Dashboards Kibana intégrant des visualisations (comme des tableaux, des graphiques et des cartes) qui permettent de répondre à certaines questions sur les données

Pour réaliser cela, j'ai utilisé des données factices de vols d’avions de lignes sur les 2 derniers mois.
